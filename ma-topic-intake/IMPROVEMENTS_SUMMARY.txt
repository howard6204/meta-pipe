┌──────────────────────────────────────────────────────────────────────────────┐
│ 2. AI SELF-CHECK PROMPTS (50+ throughout skill)                             │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   After EACH PICO element, AI asks itself:                                  │
│   ─────────────────────────────────────────────────────────────────────────│
│   After Population:                                                         │
│   ✓ Is this too narrow? (e.g., "adults 65-70" → very few studies)          │
│   ✓ Will this have enough studies? (common → yes; rare → no)               │
│   ✓ Should I warn user now? (don't wait until Phase 3!)                    │
│                                                                              │
│   After Intervention:                                                       │
│   ✓ Is this too vague? (e.g., "therapy" → specify CBT)                     │
│   ✓ Is this established enough? (FDA approved? or experimental?)           │
│   ✓ Will studies exist? (new drug <1 yr → unlikely)                        │
│                                                                              │
│   After Outcome:                                                            │
│   ✓ Is this measurable? (e.g., "feeling better" → no)                      │
│   ✓ Is this commonly reported? (quick WebSearch to verify)                 │
│   ✓ Can this be pooled? (same scales across studies?)                      │
│                                                                              │
│   ✅ Result: AI catches problems EARLY, not late                            │
└──────────────────────────────────────────────────────────────────────────────┘

 🎯 SUCCESS CRITERIA
═══════════════════════════════════════════════════════════════════════════════

✅ AI successfully guides user when:
   • User ends with feasible topic (14+/16 score)
   • Red flags caught early (not discovered later)
   • User understands WHY revisions needed
   • TOPIC.txt includes feasibility metadata
   • 4-hour assessment recommended

❌ AI fails when:
   • User starts extraction, discovers <3 studies
   • User discovers outcome not reported (should've checked)
   • High heterogeneity makes pooling impossible (should've warned)
   • Recent review makes work redundant (should've searched)
